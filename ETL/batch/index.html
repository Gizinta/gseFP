Sync Engine batch Folder

Folder that contains batch processing tools for campus-wide processing steps

Contents

This folder contains an example multiprocessing python script that cna be used to run many drawings in parallel on the same machine. This is an advanced tool that is not used as part of the basic setup.

Note that you can add additional batch processing files to this folder. The normall way to do this is create a subfolder for our process (i.e., Routing) and add the content in the folder.
